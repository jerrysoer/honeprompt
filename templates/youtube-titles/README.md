# YouTube Titles

Generate click-worthy YouTube video titles that balance curiosity with honesty.

**Category:** creators
**Author:** HonePrompt Team
**Estimated Cost:** $1-3 (25 iterations)
**Tags:** youtube, titles, seo, creators, video

## What This Template Does

Optimizes a YouTube title-writing prompt. The prompt takes a video topic and content description as input and generates a single title under 60 characters that maximizes click-through rate without resorting to clickbait. Balances curiosity, searchability, and honest representation of the content.

## Scoring Criteria

- **Click-worthiness (35%)** — Does it create enough curiosity to earn a click?
- **Accuracy (25%)** — Does it honestly represent the content?
- **Conciseness (20%)** — Is it under 60 characters with every word earning its place?
- **Searchability (20%)** — Does it contain natural search terms for the topic?
