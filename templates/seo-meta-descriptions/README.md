# SEO Meta Descriptions

Write search-optimized meta descriptions under 160 characters with keyword integration and click appeal.

**Category:** marketers
**Author:** HonePrompt Team
**Estimated Cost:** $1-3 (25 iterations)
**Tags:** seo, meta-description, search, content-marketing, serp

## What This Template Does

Optimizes a system prompt that writes SEO meta descriptions from page titles, content summaries, and target keywords. The optimizer balances the technical constraint (under 160 characters, keyword placement) with the human element (compelling enough to click in search results).

## Scoring Criteria

- **Character Length (25%)** — Under 160 characters, ideally under 155 for safety margin
- **Keyword Integration (25%)** — Target keyword placed naturally in the first half
- **CTA / Value Proposition (20%)** — Clear reason to click through to the page
- **Specificity & Accuracy (20%)** — Concrete details, not generic descriptions
- **Voice & Readability (10%)** — Active voice, natural reading in SERP context
