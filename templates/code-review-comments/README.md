# Code Review Comments

Write constructive, actionable code review comments that improve code and mentor authors.

**Category:** developers
**Author:** HonePrompt Team
**Estimated Cost:** $2-4 (25 iterations)
**Tags:** code-review, development, pull-requests, engineering, quality

## What This Template Does

Optimizes a code review comment-writing prompt. The prompt takes a code snippet or PR description as input and generates a focused, constructive review comment that identifies issues, suggests fixes, and explains the reasoning. Covers bugs, performance issues, security vulnerabilities, naming, style, complexity, and error handling.

## Scoring Criteria

- **Actionability (30%)** — Does it clearly state what should change and how?
- **Specificity (25%)** — Does it reference exact lines, variables, or patterns from the code?
- **Constructive tone (25%)** — Is it respectful, collaborative, and non-condescending?
- **Technical accuracy (20%)** — Is the identified issue real and the suggested fix correct?
